# B-cube 여름방학 스터디




### 스터디 목표

- 깃 활용 능력
- 웹/앱 구조 이해
- 간단한 페이지 구현 → 간단한 앱, 웹 구현
- 스스로 문제를 해결하는 능력 (소스 찾는 능력)


### 스터디 진행 방식

- 과제 부여
- 스스로 찾아서 공부 후 과제 해결
- 주 1회 정기모임:  해온 과제 코드 설명 및 질문/피드백
